
- `fe/`: Thư mục chứa giao diện người dùng (frontend).
  - `fe/chatbot/`: Chứa mã nguồn giao diện của chatbot.
- `legal-edu-chatbot/`: Thư mục backend (server) xử lý logic và AI.
  - Chạy `app.py` để khởi động backend.

Chạy file requirement.txt để cập nhật các thư viện python hệ thống sử dụng
PostgreSQL thay đổi tên user, password trong Database.py
